1. To convert our edge and nodes file to network file.

--> netconvert --node-files = nodes.nod.xml --edge-files = edge.edg.xml -o network.net.xml

--> Converting Right hand side network file to left hand side network file

    netconvert -s right_network.net.xml --flip-y-axis -o left_network.net.xml

    
2. Generating randomTrips file

--> python randomTrips.py -n network.net.xml -e 200 -o trip.trips.xml

3. Generating route file

--> duarouter -n network.net.xml --route-files trip.trips.xml -o route.rou.xml --ignore-errors
